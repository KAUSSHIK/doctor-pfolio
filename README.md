# MH Shriramanathan — Portfolio

Personal portfolio website for MH Shriramanathan, MBBS student and aspiring surgeon.

## Stack

Plain HTML, CSS, and JavaScript — no build step. Drop into any static host.

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Structure

- `index.html` — page markup and content
- `styles.css` — design system, layout, responsive rules
- `script.js` — mobile nav toggle, scroll reveal, year stamp

## Deploying

Compatible with GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static host. Point the host at the repo root once a domain is purchased.

## Editing content

All copy lives in `index.html`. Update sections in place — no templates or frameworks involved.
